# 그린버거 팀 오구 프로젝트

유용한 플러그인

-   bracket pair : 중괄호 마다 색으로 구분해주는거
-   auto close tag
-   colonize : 쉬프트 엔터 시 타이핑 중간에서 맨뒤로 세미콜론, 알트 엔터 시 중간에서 맨뒤에 세미콜론 찍고 다음줄로
-   prettier : 오토 포매팅 코드 깔끔하게 해주는것

CSS 사용시

-   root와 var 사용으로
-   자주쓰는거 한군데 모아놓기
